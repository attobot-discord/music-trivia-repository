# AttoBot Music Trivia Repository

This is the repository for the Music Trivia (`a!music-trivia`) feature of AttoBot that contains a list of songs for that feature.

If you want to contribute to this list, simply make a pull request and we will put that song in AttoBot!

**Notice:** NSFW songs aren't allowed. Any pull requests with NSFW songs will get declined.

Thanks for your contribution!
